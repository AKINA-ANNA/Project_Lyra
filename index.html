Local Agent Setup Wizard

Follow these quick steps to set up VirtualBox and the FastAPI Agent

Step 1: Install Oracle VirtualBox

Download and install VirtualBox on your Windows host. Ensure VBoxManage.exe is in standard folders.

Step 2: Import Master VMs

Import or build template VMs, naming them with suffix -Master (e.g. Kali-Master, Ubuntu-Master).

Step 3: Create 'GoldenMaster' Snapshot

For each Master VM, take a snapshot named exactly GoldenMaster. This is required for deployment verification.

Step 4: Launch VM Portal Agent

Run pip install -r requirements.txt inside the agent directory, then start with python main.py.
